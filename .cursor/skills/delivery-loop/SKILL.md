This is the primary skill for execution tasks.

## Required sequence

1. Plan first for development tasks.
2. Implement minimal scoped changes.
3. Run required tests/checks.
4. Produce OpenClaw-ready completion report.

## Non-negotiables

- No plan bypass for development tasks.
- No SUCCESS status without passing validation.
- Report blockers explicitly with next steps.
